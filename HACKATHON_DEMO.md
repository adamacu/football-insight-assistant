## âœ… What We Built

A conversational AI-powered football analytics tool that demonstrates professional-level sports data visualization and insights through a simple chat interface.

### Key Features (MVP)

1. **Conversational Interface**
   - Chat with football data using natural language
   - 3 demo queries working:
     - "Show me shot map for Liverpool vs Chelsea"
     - "Compare Salah vs Haaland"
     - "Analyze Arsenal"

2. **Shot Map Visualization**
   - Football pitch drawing with shot locations
   - Color-coded shots (gold = goals, red = misses)
   - Team statistics comparison (xG, shots, goals)
   - Player identification per shot

3. **Player Comparison**
   - Text-based player analysis with insights
   - Similarity scores and contextual explanations
   - Performance metrics comparison

4. **Responsive Design**
   - Dark mode interface
   - Mobile-friendly layout
   - Clean, professional UI
